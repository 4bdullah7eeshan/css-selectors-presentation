Origins of CSS

Håkon Wium Lie, a Norwegian web pioneer, first proposed the concept of CSS.

This was in 1994 while he was working at CERN alongside Sir Tim Berners-Lee.

Håkon Wium Lie later moved to W3C (World Wide Web Consortium), where he worked with others, including Bert Bos, to develop and refine the CSS specification

Universal Selectors

Because when you're not picky, everyone's invited to the styling party!

The universal selector will select all elements regardless of the type.

Hence the name “universal" 😉!

Class Selectors

A class selector will select all elements of the given class.

What is a class...you ask? 🤔

Class

A class is an HTML attribute that we can put on an HTML element.

Multiple HTML elements can have a same class.

Adding A Class To An HTML Element

							
<div class="quote">Success is 1% inspiration, 99% not getting distracted by your phone.</div>

Syntax

							
								.class-name {

								}

ID Selectors

An ID Selector will select an HTML element of the given ID.

An ID is an HTML attribute that we can put on an HTML element just like class.

IDs are unique—just like you! 😘

Grouping Selectors

For when you want to style everyone, but you just can't be bothered to do it one by one!

What if two or more groups of elements share same style(s)?

								
<!DOCTYPE html>
<html lang="en">
	<head>
		<meta charset="UTF-8">
		<title>Document</title>
	</head>
	<body>
		<h1>Zing!</h1>
		<p>I’m on a seafood diet. I see food, and I eat it.</p>
	</body>
</html>

In such cases we can use Group Selectors which essentially allows us to group our elements together.

Syntax

							
								first-element,
								second-element,
								third-element {

								}

Illustration

								
<!DOCTYPE html>
<html lang="en">
	<head>
		<meta charset="UTF-8">
		<title>Document</title>
	</head>
	<body>
		<h1>Zing!</h1>
		<p>I’m on a seafood diet. I see food, and I eat it.</p>
	</body>
</html>
								
							
								
<!DOCTYPE html>
<html lang="en">
	<head>
		<meta charset="UTF-8">
		<title>Document</title>
		<style>
			h1,
			p {
				background-color: blue;
				color: aqua;
			}
		</style>
	</head>
	<body>
		<h1>Zing!</h1>
		<p>I’m on a seafood diet. I see food, and I eat it.</p>
	</body>
</html>

Combinators

Some HTML elements have a family like relationship between each other.

								
<!DOCTYPE html>
<html lang="en">
	<head>
		<meta charset="UTF-8">
		<title>Document</title>
	</head>
	<body>
		<div class="parent">
			Parent
			<div class"child">Child One.</div>
			<div class"child">Child Two.</div>
			<div class"child">Child Three.</div>
		</div>
		<div class"child">I am not one of them.</div>
	</body>
</html>

To style such HTML elements we use Combinators.

Types of Combinators

There are four combinators:

  1. Descendant Combinators
  2. Child Combinator
  3. Adjacent/Next Sibling Combinator
  4. General Sibling Combinator

Descendant Combinator

Syntax

							
								parent child {

								}

They combine two selectors such that elements matched by the second selector are selected if they have an ancestor (parent, parent's parent, parent's parent's parent, etc.)

								
<!DOCTYPE html>
<html lang="en">
	<head>
		<meta charset="UTF-8">
		<title>Document</title>
		<style>
			.parent .child {
				color: lightgoldenrodyellow;
				background-color: darkred;
			}
		</style>
	</head>
	<body>
		<div class="parent">
			Parent
			<div class"child">Child One.</div>
			<div class"child">Child Two.</div>
			<div class"child">Child Three.</div>
		</div>
		<div class"child">I am not one of them.</div>
	</body>
</html>
								
							
								
<!DOCTYPE html>
<html lang="en">
	<head>
		<meta charset="UTF-8">
		<title>Document</title>
		<style>
			.parent .grand-child {
				color: cornflowerblue;
				font-size: 48px;
			}
		</style>
	</head>
	<body>
		<div class="parent">
			Parent
			<div class"child">Child One.</div>
			<div class"child">
				Child Two.
				<div class"grand-child">I am a grand-child!</div>
			</div>
			<div class"child">Child Three.</div>
		</div>
		<div class"child">I am not one of them.</div>
	</body>
</html>

Syntax

						
							selector::pseudo-element {

::first-line

It is used to add a special style to the first line of a text.

Illustration

								
<!DOCTYPE html>
<html lang="en">
	<head>
		<meta charset="UTF-8">
		<title>Document</title>
		<style>
			p::first-line {
				color: seagreen;
				font-variant: small-caps;
				font-weight: bold;
				font-size: 24px;
			}
		</style>
	</head>
	<body>
		<p>Once, a squirrel tried to crack a nut with a hammer. After several failed attempts, a wise old owl said, "Why not use your teeth?" The squirrel replied, "I wanted to do it with style!"</p>
	</body>
</html>

::first-letter

It is used to add a special style to the first letter of a text.

Illustration

								
<!DOCTYPE html>
<html lang="en">
	<head>
		<meta charset="UTF-8">
		<title>Document</title>
		<style>
			p::first-letter {
				color: blue;
				font-size: xx-large;
			}
		</style>
	</head>
	<body>
		<p>My bed is a magical place I suddenly remember everything I forgot to do.</p>
		<p>Why don’t skeletons fight each other? They don’t have the guts.</p>
		<p>Parallel lines have so much in common. It’s a shame they’ll never meet.</p>
	</body>
</html>

::before

It is used to insert some content before the an element.

Illustration

								
<!DOCTYPE html>
<html lang="en">
	<head>
		<meta charset="UTF-8">
		<title>Document</title>
		<style>
			h1::before {
				content: "\1F525";
			}
		</style>
	</head>
	<body>
		<h1>Welcome!</h1>
	</body>
</html>

::after

It is used to insert some content after the an element.

Illustration

								
<!DOCTYPE html>
<html lang="en">
	<head>
		<meta charset="UTF-8">
		<title>Document</title>
		<style>
			li::after {
				content: '✅';
			}
		</style>
	</head>
	<body>
		<h1>A Day In My Life</h1>
		<ul>
			<li>Brush my teeth with a toothbrush made of chocolate</li>
			<li>Walk the dog to the moon and back</li>
			<li>Finish reading a book I started in 1999</li>
			<li>Sing karaoke to my plants</li>
			<li>Learn to speak fluent dolphin</li>
		</ul>
	</body>
</html>

::selection

matches the portion of an element that is selected by a user.

Illustration

								
<!DOCTYPE html>
<html lang="en">
	<head>
		<meta charset="UTF-8">
		<title>Document</title>
		<style>
			::selection {
				color: white;
				background-color: black;
			}
		</style>
	</head>
	<body>
		<h1>Is this trippy for you?! You sure?!</h1>
	</body>
</html>
